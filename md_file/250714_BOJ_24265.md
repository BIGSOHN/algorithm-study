**문제 링크**: https://www.acmicpc.net/problem/24265

**오늘의 메모(선택)**: 조합 구하기 공식을 몰라서 찾았다.

**코드(선택)**:

```c++
#include <iostream>

using namespace std;

int main() {
	long long n;

	cin >> n;
	cout << n * (n - 1) / 2 << '\n';
	cout << "2\n";

	return (0);
}
```
