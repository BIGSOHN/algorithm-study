**오늘의 메모(선택)**: 범위 체크해서 overflow나는거 확인했어야 했는데 다음부터 입력값이 뭔지 생각해야겠다.

참고 :

**코드(선택)**:

```c++
#include <iostream>

using namespace std;

long long factorial(int n) {
	if (n <= 1) {
		return (1);
	}
	return (n * factorial(n - 1));
}

int main() {
	int N;

	cin >> N;
	long long result = factorial(N);
	cout << result << '\n';

	return (0);
}
```
