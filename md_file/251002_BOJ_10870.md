**문제 이름 또는 번호**: 10870 / 피보나치 수5

**문제 링크**: https://www.acmicpc.net/problem/10870

**오늘의 메모(선택)**: 피보나치는 너무 어렵다

**코드(선택)**:

```c++
#include <iostream>

using namespace std;

int fibonacci(int n) {
	if (n == 0) {
		return (0);
	}
	if (n == 1) {
		return (1);
	}
	return (fibonacci(n - 1) + fibonacci(n - 2));
}

int main() {
	int N;
	cin >> N;
	cout << fibonacci(N) << '\n';
	return (0);
}
```
